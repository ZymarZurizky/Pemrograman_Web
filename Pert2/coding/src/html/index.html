ABOUT ME

Perkenalkan, nama saya Zidan Ammar Zurizky dengan NIM 20230801398, mahasiswa dari Program Studi Teknik Informatika di Universitas Indonusa Esa Unggul. Saat ini saya berada di semester kedua dan sedang mengikuti mata kuliah Pemrograman Web, yang menjadi salah satu bidang yang sangat saya minati dalam dunia informatika.

Latar belakang pendidikan saya sebelumnya adalah dari jurusan Rekayasa Perangkat Lunak (RPL) saat di SMK, di mana saya telah dikenalkan dengan dasar-dasar pemrograman dan pengembangan perangkat lunak. Pengalaman tersebut menjadi pondasi awal bagi saya untuk lebih mendalami dunia pengembangan web, khususnya bagaimana membangun aplikasi berbasis web yang tidak hanya menarik dari segi tampilan, tetapi juga berfungsi dengan baik dari sisi sistem.

Sebelum kuliah, saya sudah memiliki dasar pengetahuan di bidang pemrograman karena latar belakang saya dari SMK dengan jurusan Rekayasa Perangkat Lunak (RPL). Dari situ, saya terbiasa menggunakan bahasa pemrograman seperti HTML, CSS, JavaScript, dan PHP secara dasar. Namun, saya menyadari bahwa masih banyak hal yang perlu saya pelajari lebih dalam, terutama dalam membangun aplikasi web yang terstruktur, responsif, dan fungsional.

Melalui mata kuliah ini, saya berharap dapat meningkatkan kemampuan teknis saya di bidang pengembangan web, memahami alur kerja front-end dan back-end dengan lebih baik, serta mampu membuat proyek web sederhana sebagai bentuk implementasi dari materi yang telah dipelajari.
